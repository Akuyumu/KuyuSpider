# Design of Power Transmission
## Information needed
- Nature of driven machine
- Level of power to be transmitted
- Desired output speed
- Rotational speed of input
## Steps involved
- Selecting suitable machine element
- Estimating satisfactory machine element size
- Evaluate machine element performance against requirements
- Iterating until performance meets requirements
